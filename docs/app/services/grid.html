Located a point in the actual grid system.

Steps: 1. Invert the point coords to fullfil our notation system (rows, colss) 2. First substract the padding => to find the coordinate of the point

in relation to the Origin of the Grid system

  1. Divide by the grid size ([height, width]) and ‘floor’ the number
