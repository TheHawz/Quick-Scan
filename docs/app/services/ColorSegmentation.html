ColorSegmentation¶

Module for Color Segmentation functions.

Used for segmenting a color object of a frame, and improving the resultant mask.

Author: Lara Blanco Freire Modified by: Pablo Losada Rodríguez

app.package.services.ColorSegmentation.getColorMask(img, bottom_thres, top_thres)¶

Returns a mask with the pixels whose values are between the thresholds.

Parameters

  • img (np.ndarray, dtype=uint8) – Image for processing.

  • bottom_thres (list) – Threshold in HSV form.

  • top_thres (list) – Threshold in HSV form.

Returns

output array of the same size as img and CV_8U type

Return type

np.ndarray
